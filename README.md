# Compras Estupidas de Elmer.

Catalogo personal para organizar comics, mangas, figuras y otras compras futuras.

## Funciones

- Agregar items pendientes o conseguidos.
- Guardar formato, prioridad, tienda, valor total, notas e imagen.
- Usar imagen por URL o buscar fotos con Google Custom Search.
- Filtrar, buscar, ordenar, editar y eliminar items.
- Exportar e importar el catalogo en JSON.

## Uso

Abre `index.html` en el navegador o publica esta carpeta como sitio estatico.

Los datos se guardan en `localStorage`, dentro del navegador donde uses la app.
